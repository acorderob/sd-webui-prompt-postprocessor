# ACB PPP Wildcard Concat node

Concatenates up to 10 selected wildcards.

## Inputs

* **previous_prompt**: Previous prompt to add to.
* **filter**: Filter text to reduce the available wildcard keys in the wildcard_N selectors.
* **separator**: Separator string to join the selected wildcards.
* **wildcard_1** to **wildcard_10**: Wildcard selectors.

## Outputs

* **prompt**: The resulting prompt.
